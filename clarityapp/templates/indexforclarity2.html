Данный сервис предоставляет возможность получать и анализировать данные по рекламной кампании по API из кабинета Яндекс Директ и автоматически сортировать и группировать их для анализа. Для настройки понадобится: токен, который нужно получить от Яндекс Директ и наименование кампании, для которой требуется анализ данных, период времени, за который требуется анализ.

На сегодняшний день доступны следующие отчеты:

  • Позитивная конверсия: отчет, содержащий ключевые фразы, которые принесли заявки, с низкой стоимостью клика (от 50 руб/клик и ниже).
  • Отрицательная конверсия: отчет, содержащий ключевые фразы, которые не принесли заявки, с высокой стоимостью клика (от 100 руб/клик и выше).
